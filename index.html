MLCourseProject

This Project is part of the Coursera Practical Machine Learning Course, It is about Utilizing the concepts learnt during course and use it for predicting the outcome for the given Datasets.

DataSet Background

Using devices such as Jawbone Up, Nike FuelBand, and Fitbit it is now possible to collect a large amount of data about personal activity relatively inexpensively. These type of devices are part of the quantified self movement â€“ a group of enthusiasts who take measurements about themselves regularly to improve their health, to find patterns in their behavior, or because they are tech geeks. One thing that people regularly do is quantify how much of a particular activity they do, but they rarely quantify how well they do it. In this project, goal is to use data from accelerometers on the belt, forearm, arm, and dumbell of 6 participants. They were asked to perform barbell lifts correctly and incorrectly in 5 different ways. More information is available from the website here: http://groupware.les.inf.puc-rio.br/har (see the section on the Weight Lifting Exercise Dataset).

The Approach

I have a Developed a R code named (myPrediction.R), the code includes two functions namely:
1. myPrediction( ) - Takes two arguments filenames of Training data and Test Data respectively
2. predSelect( ) - Takes a Data frame as an argument and filters out the unnecessary predictors.
I have reduced the predictors from 160 to 56 by filtering out columns with large number of NA's and taking into consideration only columns of type numeric. I went with this approach mainly because RandomForest splits better with numeric data and produces better results.
I have split the Training Data using createDatapartition function in R and subsetted 70% for training and 30% for testing.

Model

I have used RandomForest Algorithm to train the Data and used classe as outcomes and remaining columns as predictors. Using this model I predicted the 30% test data and obtained 99.993% accuracy. The plot below shows the error rates for each tree created by RandomForest model.

Finally I applied this model on Test Dataset which contained 20 samples and predicted their outcomes, I submitted the files generated in Coursera.
